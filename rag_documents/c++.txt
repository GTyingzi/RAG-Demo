面向对象：C++ 是一种面向对象的语言，支持封装、继承和多态等面向对象的基本特征
模板：C++ 支持编写泛型代码，使用模板可以有效地处理各种数据类型
强类型：C++ 是一种强类型语言，在编译时对变量的类型进行了严格的检查
兼容 C 语言：C++ 基于 C 语言，并对 C 语言的代码做了扩展。C++ 程序员可以方便地使用 C 语言代码
高效率：C++ 是一种高效率的语言，特别是面向对象编程和模板编程的应用使 C++ 更加高效
多平台：C++ 是一种跨平台语言，可以在多种操作系统，如 Windows、Linux、macOS 等上运行
强大的库：C++ 拥有强大的标准库，可以提供各种常用的功能，如字符串处理、数学运算、文件输入/输出等
